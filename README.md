# 📊 Consumer Electronics Sales Analytics & Customer Insights System

## 🚀 Project Overview
This project analyzes a large-scale consumer electronics sales dataset to extract meaningful business insights.

## 🧰 Tools Used
- SQL
- Python
- Power BI
- Excel

## 📂 Dataset Overview
- 50K+ Customers
- 200K+ Orders
- 1.5M Products Sold
- 100K Reviews

## 📊 Key Insights
- Smartphones are the top revenue category
- High-value customers contribute significantly to revenue
- Revenue peaks in July
- Customer satisfaction is stable around 3.0 rating

## 📈 Dashboards
Power BI dashboard provides:
- Revenue analysis
- Customer insights
- Product performance
- Market trends

## 🧠 Business Recommendations
- Improve high-revenue brand satisfaction
- Target VIP customers with loyalty programs
- Increase marketing for high-rated products

## 🗂️ Project Structure
- SQL Analysis
- Python Data Generation
- Power BI Dashboard
- ERD Design
- Documentation
